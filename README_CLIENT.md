# Thar Pathology Laboratory Billing System

## Quick Start
1. Double-click `TharPathologyLab.exe`
2. Use these login details:
   - Username: `user`
   - Password: `user1234`
   For admin:
   username: admin
   password: admin123
3. The system will automatically create a new database

## Features
- Patient registration and billing
- Test management  
- Professional receipt printing
- Revenue reports
- User management

## System Requirements
- Windows 7/8/10/11
- 4GB RAM
- 500MB free space



## Important Notes
- The database file (lab_billing.db) will be created automatically
- Keep the .exe file in a folder where you have write permissions
- No installation required - just run the .exe file
